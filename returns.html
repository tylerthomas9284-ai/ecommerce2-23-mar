Return Policy

Transparent returns guidance for eligible purchases.

This page can be used as the brand’s return policy reference for USA customers.

Return window and conditions

  • Eligible unworn items may be returned within 14 days of delivery.
  • Returned products must include original packaging where applicable and show no signs of wear or damage.
  • Final sale items, personalized products, pierced earrings, and hygiene-sensitive items are not eligible for return unless required by law.
  • Contact customer service before sending a return so the request can be reviewed and routed correctly.
